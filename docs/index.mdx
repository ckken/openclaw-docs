# OpenClaw 使用手册

欢迎使用 OpenClaw！本手册提供 OpenClaw 的完整配置参考和使用指南。

## 什么是 OpenClaw？

OpenClaw 是一个 AI 助手框架，支持多通道、多模型的智能代理系统。

## 主要特性

- 🤖 **多 Agent 支持** - main、coder、watchman 等多种代理
- 📱 **多通道集成** - Telegram、Discord、Slack 等
- 🔗 **灵活绑定** - 消息自动路由到对应 Agent
- 🔒 **安全控制** - 细粒度的工具权限管理
- 📊 **监控告警** - 完整的系统监控能力
