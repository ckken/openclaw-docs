# 环境变量

OpenClaw 支持通过环境变量进行配置。

## 常用环境变量

### OpenClaw 配置

| 变量 | 说明 |
|------|------|
| `OPENCLAW_HOME` | OpenClaw 主目录 |
| `OPENCLAW_CONFIG` | 配置文件路径 |

### 模型配置

| 变量 | 说明 |
|------|------|
| `OPENAI_API_KEY` | OpenAI API Key |
| `ANTHROPIC_API_KEY` | Anthropic API Key |
| `MINIMAX_API_KEY` | MiniMax API Key |

### 日志配置

| 变量 | 说明 |
|------|------|
| `OPENCLAW_LOG_LEVEL` | 日志级别 (silent/fatal/error/warn/info/debug/trace) |
| `OPENCLAW_LOG_FILE` | 日志文件路径 |

### 调试

| 变量 | 说明 |
|------|------|
| `DEBUG` | 调试模式 |
| `VERBOSE` | 详细输出 |

## 使用示例

```bash
# 设置日志级别
export OPENCLAW_LOG_LEVEL=debug

# 设置 API Key
export MINIMAX_API_KEY=your-key

# 指定配置目录
export OPENCLAW_HOME=/custom/path
```
